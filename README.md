# intelfyai

**IntelF Token (INTF)** is a fixed-supply, tax-free BEP-20 token on **BNB Smart Chain (BSC)**.

It powers the **intelfyai** AI copilot ecosystem designed for e-commerce, Shopify and algorithmic trading.
